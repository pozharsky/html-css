# Псевдоклассы: динамические

</div>

<br />

**Динамические псевдоклассы** — применяют стили к элементам, когда происходит определенное событие.

События могут быть следующими:
- наведение мыши;
- нажатие мыши;
- установка фокуса;
- посещение ссылки.



<br />

<div align="center">

## `hover`

</div>

**Псевдокласс `hover`** — применяет стили к элементу, если на него навели курсор мыши.

**Пример 1.1.** Написание псевдокласса `hover`.
```html
<p>Текст</p>
```

```css
/* Эти стили будут применены при наведении курсора мыши на параграф */
p:hover {
  color: tomato;
}
```



<br />

<div align="center">

## `active`

</div>

**Псевдокласс `active`** — применяет стили, когда пользователь активирует элемент. Под активацией подразумевается время между нажатием и отпусканием кнопки мыши на элементе.

**Пример 2.1.** Написание псевдокласса `active`.
```html
<p>Текст</p>
```

```css
/* Эти стили будут применены при нажатии мыши на параграфе */
p:active {
  color: tomato;
}
```



<br />

<div align="center">

## `focus`

</div>

**Псевдокласс `focus`** — применяет стили, когда пользователь устанавливает фокус на элементе.

Фокус можно установить двумя путями:
1. Нажать на элемент.
2. Переместиться к элементу с помощью клавиши TAB.

**Пример 3.1.** Написание псевдокласса `focus`.
```html
<input type="text">
```

```css
input:focus {
  background: tomato;
}
```
